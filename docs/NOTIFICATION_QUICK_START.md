## üîß Troubleshooting

### Not Receiving Notifications?

**Check Permissions:**
1. Go to Profile > Notifications
2. Verify "Notification Permission" shows "Granted"
3. If not, tap "Request Permissions" again

**Check Device Settings:**
1. Open your device Settings
2. Find Muslim Life Hub
3. Ensure notifications are enabled
4. Check Do Not Disturb mode

**Check Scheduled Notifications:**
1. Go to Profile > Notifications
2. Look at "Scheduled Notifications" count
3. Should show a number greater than 0
4. If 0, try toggling notification types off and on

### Location Not Working?

1. Ensure location permission is granted
2. Check if location services are enabled on your device
3. Try restarting the app

### Notifications Delayed?

Some Android devices delay notifications to save battery:
1. Go to device Settings
2. Find Battery or Power Management
3. Disable battery optimization for Muslim Life Hub
